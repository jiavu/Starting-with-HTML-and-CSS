    Flexible Boxes: a flex-container

    To fit a box to the size of several contents of different sizes use a flexbox:
    .flex-container {
    display: flex;
    flex-wrap: nowrap;
    }
    Using a Flexbox rather then placing by float property +width is a more modern way of creating column layouts.
